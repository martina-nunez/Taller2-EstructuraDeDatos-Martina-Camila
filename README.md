#Descripción:
    Este proyecto es un reproductor de musica hecho en C++, permite cargar canciones desde un archivo de texto, almacenarlas y gestionar su reproducción, implementando distintas estructuras de datos como listas con enlaces dobles, uso de árboles trie y uso de Heap, para implementar las distintas funciones.

#Para compilar:
    g++ core/main.cpp core/SistemaImpl.cpp classes/Cancion.cpp data_structures/List.cpp data_structures/Node.cpp classes/Reproductor.cpp data_structures/HeapNode.cpp data_structures/Heap.cpp data_structures/CancionTrie.cpp data_structures/NodoTrie.cpp data_structures/NodoCancion.cpp data_structures/Trie.cpp -o .\iniciar

#Para ejecutar:
    .\iniciar
