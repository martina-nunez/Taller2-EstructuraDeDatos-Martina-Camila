#Reproductor

#Integrantes:
    Martina Núñez

#Descripción:
    Este es un programa con el cual se pueden reproducir una lista de canciones, mediante el uso de la consola, se pueden realizar diferentes acciones, como cambiar la cancion a la siguiente y a la anterior, agregaro eliminar canciones, activar el modo aleatorio y las repeticiones de las canciones, entre otras cosas.

#Para compilar:
    g++ core/main.cpp core/SistemaImpl.cpp classes/Cancion.cpp data_structures/List.cpp data_structures/Node.cpp classes/Reproductor.cpp -o iniciar

#Para ejecutar:
    .\iniciar
